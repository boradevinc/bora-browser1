# Supported Platforms

This page lists platforms officially supported by bora-browser, and their associated repositories.

* Android: [bora-browser-android](https://github.com/ungoogled-software/bora-browser-android)
* Arch Linux: [bora-browser-archlinux](https://github.com/ungoogled-software/bora-browser-archlinux)
* Debian, Ubuntu, and derivatives: [bora-browser-debian](https://github.com/ungoogled-software/bora-browser-debian)
* Fedora and CentOS: [bora-browser-fedora](https://github.com/ungoogled-software/bora-browser-fedora)
* Portable Linux (for any Linux distro): [bora-browser-portablelinux](https://github.com/ungoogled-software/bora-browser-portablelinux)
* Windows: [bora-browser-windows](https://github.com/ungoogled-software/bora-browser-windows)
* macOS: [bora-browser-macos](https://github.com/ungoogled-software/bora-browser-macos)
